System Requirement

Hardware Specification:
The implementation of the proposed system requires the following hardware specifications.
          a. CPU Type : Pentium or Above.
          b. CPU Clock : 2.6 ghz or above
          c. RAM : 1 GB or above
          d. CD Drive : Any make
          e. U.P.S.[5 Kv ]: Any Make
Software Specification:
The software is designed under the following environment:
          Language : Java EE
          Scripting Language : HTML, Java Script
          Database : SQL-Server
          Operating System : Windows 7/8
          Web Server : Tomcat
